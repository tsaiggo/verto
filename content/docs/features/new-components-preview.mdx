## Embed

`<Embed>` resolves provider metadata at build time and renders a static `.link-card` variant — no iframes, no client JS. See the [full Embed reference](/read/docs/features/embed).

Author-overridden card (no network call):

<Embed
  url="https://example.com/article"
  title="Article title (manually overridden)"
  description="When any override is supplied, Embed skips the network entirely — useful for private URLs or when you want exact wording."
  siteName="Example"
/>

Forced bookmark renderer (degrades cleanly if metadata can't be fetched):

## D2

`<D2>` mirrors `<Mermaid>` exactly — fenced code block or explicit component. The WASM bundle (~3 MB) is dynamic-imported so this page is the only place that pays the cost. See the [full D2 reference](/read/docs/features/d2).

```d2
direction: right
authors -> file: write
file -> verto: build
verto -> reader: serve
```

Theme-aware: toggle dark mode and watch the diagram swap palettes automatically.
