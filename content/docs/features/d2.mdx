# Diagrams (D2)

Verto supports [D2](https://d2lang.com) diagrams in the same two equivalent forms as Mermaid.

## As a fenced code block

```d2
direction: right
authors -> file: write
file -> verto: build
verto -> reader: serve
```

A `rehype-d2` plugin intercepts these blocks before Shiki sees them and routes them to the client-side D2 renderer.

## As an MDX component

For when you want to mix props, conditionals, or data binding into the source:

<D2 chart={`
shape: sequence_diagram
reader -> verto: GET /read/docs/intro
verto -> fs: read intro.md
fs -> verto: source
verto -> reader: rendered HTML
`} />

## Performance and safety

The D2 WASM bundle (~3 MB) is **dynamic-imported** — pages without D2 diagrams pay zero bundle cost. The rendered SVG is sanitized through DOMPurify before being injected into the DOM, so even an attacker-crafted D2 source cannot smuggle script tags into the page.

## Theming

Pass `themeId` and `darkThemeId` props to pick from D2's [built-in themes](https://d2lang.com/tour/themes). Defaults are the neutral light theme (id `0`) and D2's dark theme (id `200`). The diagram re-renders automatically when the user toggles light / dark mode.
