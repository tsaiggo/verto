# Verto Docs

Welcome to **Verto** — a fast, MDX-native reader for your notes, docs and blog.

These pages are the documentation that ships **inside** Verto itself. They are
not read from your local folder or a connected GitHub repository — they are
embedded into the app at build time, so they are always one click away no
matter which content source you have connected.

## What's here

- **[Getting Started](/read/_docs/getting-started)** — connect a source and read your first document.
- **[Authoring](/read/_docs/authoring)** — write MDX with built-in block components.
- **[Content Sources](/read/_docs/content-sources)** — point Verto at a local folder, GitHub, or OneDrive.

> These docs live under the reserved `/read/_docs/…` path, so your own content
> can never collide with them.
